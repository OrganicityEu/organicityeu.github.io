OrganiCity Experimentation Monitoring API

Version: 1.0.0

Monitorisation of experiments taking place in OrganiCity platform

Schemes: https

Base URL: https://organicity.eu/v1

Summary

Path Operation Description
/experiments GET

retrieve the list of experiments running in the OC platform
/experiments/{experimentId} DELETE

deletes an experiment
GET

find experiment by ID
/experiments/{experimentId}/sources GET

retrieve the list of sources for a given experiment.
POST

add new sources to a given OrganiCity experiment
/experiments/{experimentId}/statistics GET

retrieve statistics for a given experiment.
